Eyes Wide Open

Creative Coding
Print
Web Design
3D

Eyes Wide Open is a book and website about a collection of security cameras in NYC. The book features a collection of pictures with accompanying text, detailing the progression of surveillance in the city.

The website features a 3D model of a security camera, and an WebXR experience that uses the NYT API to pull articles relating to surveillance.
